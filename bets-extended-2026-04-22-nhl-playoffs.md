## Ставка #E1 — СИЛЬНЕЙШАЯ

> **Монреаль — победа в серии @ 2.55**  
> Тампа-Бэй — Монреаль

### Математика

| Параметр | Значение |
|----------|---------|
| Implied probability | 39.2% |
| Fair probability (без маржи) | ~37.5% |
| Историческая база | Команда, ведущая 1–0, выигрывает серию **в 65% случаев** (НХЛ 2000–2024) |
| Поправки | Vasilevskiy под вопросом (−5% Тампа) / Монреаль моложе и голоднее (+3%) |
| **Моя оценка P(Монреаль серия)** | **~47%** |
| Преимущество | **+9.5% ✅✅** |
| Kelly full | (0.47 × 1.55 − 0.53) / 1.55 = **13.2%** |
| Kelly 25% | 3.3% → **3%** |
| **Ставка** | **13₽** |
| Потенциал | 33.2₽ |

### Обоснование

- Монреаль ведёт серию 1–0, и исторически это сильнейший предиктор итога серии
- Команда, которая берёт инициативу дома в Игре 1, психологически владеет серией
- Слафковский в горячей форме (хет-трик + OT победный гол = уникальное историческое выступление)
- Vasilevskiy пропустил тренировку — если он не выйдет и в Игре 3/4, Тампа в серьёзной опасности
- Тампа проиграла 9 из 10 последних домашних матчей плей-офф — ситуационная слабость
- **BetBoom оценивает Монреаль всего в 37.5%** при серийной ведущей позиции — явная неэффективность рынка

## Ставка #E2 — СИЛЬНАЯ

> **Лос-Анджелес Ф2 (+1.5) @ 1.85**  
> Колорадо — Лос-Анджелес

### Математика

| Параметр | Значение |
|----------|---------|
| Implied probability | 54.1% |
| Fair probability (без маржи) | ~52% (маржа ~4.3%) |
| Историческая база | Колорадо выигрывает в регулярное время с разницей ≥2 шайб примерно в **30–33%** матчей |
| Игра 1 факт | 2–1 (Kings держались до финальной сирены) |
| Мотивация LA | Wild-card без груза — играют без давления |
| **Моя оценка P(Колорадо −1.5)** | **~32%** → P(LA +1.5) = **68%** |
| Преимущество | **+16% ✅✅✅** |
| Kelly full | (0.68 × 0.85 − 0.32) / 0.85 = **30.6%** |
| Kelly 25% | 7.65% → cap **8%** |
| **Ставка** | **35₽** |
| Потенциал | 64.8₽ |

### Обоснование

- Игра 1 закончилась 2–1 — Kings продемонстрировали, что умеют держать счёт
- Колорадо — лучшая команда регулярного сезона, НО плей-офф — другой хоккей
- MacKinnon/Makar доминируют, но Kings плотно закрываются у своих ворот (Wedgewood 24 сейва)
- Фора −1.5 для фаворита @ 1.91 — это рынок говорит «50/50», а фактически нет
- Серия 2023 Kings vs Colorado длилась 7 игр — это команды с похожей плей-офф ДНК
- **Ставка выигрывает при любом счёте 1–0, 2–1, 3–2, OT** — только проигрывает при 3–0, 4–0, 4–1, 4–2 в пользу Колорадо по регулярному времени

## Ставка #E3 — ХОРОШАЯ

> **Тотал Меньше 5.5 @ 2.16**  
> Баффало — Бостон

### Математика

| Параметр | Значение |
|----------|---------|
| Implied probability | 46.3% |
| Fair probability (без маржи) | ~44.4% (маржа ~4.2%) |
| Историческая база НХЛ плей-офф | Среднее 5.0–5.3 гола/матч (регулярка: 5.8–6.2) |
| Линия БК | Основная линия **6.0** (говорит о ожидании голов), но М5.5 @ 2.16 не скорректирована |
| Игра 1 факт | 4–3 comeback — 7 голов, но эмоциональный матч (comeback с 2–3 в 3-м периоде) |
| Травмы | Thompson day-to-day, Luukkonen day-to-day, Samuelsson OUT |
| **Моя оценка P(Меньше 5.5)** | **~57%** |
| Преимущество | **+12.6% ✅✅** |
| Kelly full | (0.57 × 1.16 − 0.43) / 1.16 = **19.7%** |
| Kelly 25% | 4.9% → **5%** |
| **Ставка** | **22₽** |
| Потенциал | 47.5₽ |

### Обоснование

- БК сам выставил основную линию 6.0, что означает «нормально ожидать 6 голов» — но М5.5 @ 2.16 оценён будто шанс «меньше 6» всего 44%
- Это математическое противоречие: если линия 6.0, то М5.5 должна быть около 40–45% — и здесь оценена правильно, НО плей-офф реально даёт 52–55% на М5.5
- Если Thompson и Luukkonen под вопросом — атака Баффало ослаблена, меньше голов
- Если Luukkonen не выйдет — backup вратарь лучше держит счёт плотным (сам осторожен)
- Матч 2 часто более осторожный после эмоционального возвращения в Игре 1
